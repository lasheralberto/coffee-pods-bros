🧠 Product Spec — “Mediterranean Coffee Ritual”
1. 🎯 Objetivo

Transformar la web de e-commerce de café en una experiencia de estilo de vida mediterráneo, donde:

El usuario no compra café
El usuario elige un momento del día + una emoción
2. 🧩 Core Concept
El producto no es café en grano.

El producto es:
→ Un ritual diario mediterráneo
→ Una pausa consciente
→ Una identidad aspiracional
3. 🖥️ Estructura de la Web

1. Abre la ventana
2. Sirve el café
3. No mires el móvil
4. Respira
5. Este momento es tuyo

👉 Debe ser visual + minimalista

8. 🛒 Estrategia de Conversión
Cambios clave:
Tradicional	Nuevo
“Comprar café”	“Elegir momento”
Filtros técnicos	Filtros emocionales
Notas de cata	Sensaciones
Precio	Experiencia
9. 🚫 Anti-Patrones (NO implementar)
- Lenguaje técnico excesivo
- UI tipo catálogo de tostador
- Comparativas de origen complejas
- Enfoque en precio
10. 🧠 Insight Implementable
El usuario no quiere mejor café.

Quiere:
→ sentir control sobre su día
→ tener un pequeño ritual propio
→ vivir más lento sin cambiar su vida entera
11. 🧩 One-liner para toda la web
No vendes café.

Vendes cómo debería sentirse tu día.
